SUPER SHOOT

Simple shoot'em up implemented in Java (1.8.0_212); GWT (2.8.2) and Android (upwards 4.2.2)
using LibGDX 1.9.10.


Uploaded to Docker io; creating this public image:  docker.io/jeag2002/firsttestgdx
Deployed as EC2 container, and populated within this public ip http://3.122.232.86:80

Following the next instructions: https://reflectoring.io/aws-deploy-docker-image-via-web-console/
